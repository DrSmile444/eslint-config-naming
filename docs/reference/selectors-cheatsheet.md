
# Selectors cheatsheet

This project uses `@typescript-eslint/naming-convention` selectors like:

* `variable`
* `function`
* `memberLike`
* `parameter`
* `class`
* `interface`
* `typeLike`
* `enum`
* `enumMember`
* `objectLiteralProperty`

## Modifiers (examples)

* `public` / `private` / `protected`
* `static`
* `readonly`
* `exported`
* `global`
* `destructured`
* `requiresQuotes`

When in doubt, start from the [Rule Matrix](/rules/).
